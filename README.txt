# Ring for Homey
Connect your Ring products with Homey.
Two-Factor Security Authentication is now supported. Only the mode where it is disabled in your Ring account has been tested. In this way you receive and email with an authentication code when you try to authenticate the first time. Once you have this code enter it into the the app configuration page and authenticate again.

## Currently supports:
* Ring Chime (Pro)
* Ring Stick Up Cam
* Ring Video Doorbell (Pro)
